# AppWeather  
Приложение погоды с интерактивным видеофоном.  
В яндекс поиск первая ссылка по вводу appweather  
"AppWeather" занимает первое место в результатах поиска Яндекс по запросу "appweather".  
Использовал **Vue 3**, **Pinia**, **Canvas**.     
Для регистрация домена использовал **Рег.ру**.    
Для хостинга использовал **Спринтхост**.    
[Demo]http://appweather.ru
